## ✍️ Typography

### Font Families

#### 1. Cormorant Garamond
* **Role:** Display / Hero Headings
* **Weights:** 300–700
* **Description:** Elegant, feminine serif — used for the brand name and large titles. Evokes refinement and timelessness.
* **Sample Text:** *Lyra*

#### 2. DM Serif Display
* **Role:** Section Headings / Module Titles
* **Weights:** 400
* **Description:** Warm, readable serif for section headers. Balances the elegance of Cormorant with practicality.
* **Sample Text:** *Task Management*

#### 3. Geist (or SF Pro Display)
* **Role:** Body / UI / Labels
* **Weights:** 300–600
* **Description:** Modern, clean sans-serif for all UI text, descriptions, and labels. Pairs perfectly with the serif display fonts.
* **Sample Text:** *Your day, organized.*

#### 4. Noto Naskh Arabic
* **Role:** Arabic / Quran / Hijri
* **Weights:** 400–700
* **Description:** For all Arabic content — Quran verses, Hijri dates, Islamic module text. Elegant and readable.
* **Sample Text:** *بسم الله الرحمن الرحيم*
